## Instructions To Run Locally

Clone the project

```bash
  git clone https://github.com/dhairyaostwal/qoohoo-Internship-Task.git
```

Go to the project directory

```bash
  cd qoohoo-Internship-Task
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
